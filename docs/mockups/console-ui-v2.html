售后投诉处理流程

当客户表达强烈不满或明确要求人工服务时，Agent 应遵循以下流程。此词条与分流规则 ESCALATE_A 联动。

第一步：确认诉求

用品牌官方口吻回应，不具名客服。示例：「非常抱歉给您带来不好的体验，方便描述一下具体情况吗？」

第二步：判断升级

  • 含「投诉」「转人工」「真人」→ 触发 ESCALATE_A
  • 发送 customerLine 后 mute 24h
  • 同步告警维护者微信

第三步：知识检索优先级

检索时本词条权重 +2，优先于通用 FAQ。
