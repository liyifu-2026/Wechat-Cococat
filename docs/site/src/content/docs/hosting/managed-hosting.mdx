Don't want to manage your own infrastructure? We offer managed hosting for agent-wechat.

## What you get

- A dedicated agent-wechat instance running on our infrastructure
- Server URL and auth token — plug directly into your OpenClaw bot, Wechaty app, or custom integration
- Residential proxy pinned to your location — avoids datacenter IP detection by WeChat
- Automatic version updates — always running the latest agent-wechat release
- Monitoring and automatic restarts

## Guarantees

- **Uptime**: We monitor the container and restart it automatically if it goes down
- **Data persistence**: Your WeChat session and data persist across restarts
- **Security**: Each instance runs in an isolated environment with its own auth token

## Limitations

- WeChat sessions can still disconnect due to WeChat server-side changes — scan the QR code in WeChat if prompted
- Infrastructure updates (OS patches, Docker updates) may cause brief downtime
